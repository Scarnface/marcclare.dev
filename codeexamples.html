Netmatters

SCSS offers a great opportunity for modular code. The Netmatters' website features a responsive design which loads different headers depending on the width of the screen it is viewed in. This obviously results in some repetition as features are added or removed depending on the version being shown.

To implement this, I broke down the constituent parts of the header and created them as modules using SCSS mixins. To construct a version of the header I simply added the mixins to a further mixin containing all the required components, these are then called at the appropriate breakpoints.
